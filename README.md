3. The Deployment "Cheat Sheet"

To get this code live so you can send a working link to Frederick, follow these steps:

Option A: The Vercel "Drag & Drop" (Fastest)

Go to Vercel.com.

If you don't have an account, sign up with your GitHub account.

Navigate to the "Deploy" or "New Project" section.

Simply drag your index.html file into the deployment window.

Vercel will give you a live URL (e.g., thomas-watson-concept.vercel.app).

Option B: GitHub Pages (Best for Portfolios)

Create a new repository on GitHub named thomas-watson-surveys.

Upload your index.html.

Go to Settings > Pages.

Under Branch, select main and / (root), then click Save.

Your site will be live at https://jameswoolforddeisgn.github.io/thomas-watson-surveys-concept/.

Option C: Netlify Drop

Go to Netlify Drop.

Drag your folder containing the index.html into the box.

Instantly get a live URL to share in your pitch.
